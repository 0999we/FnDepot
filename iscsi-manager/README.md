# iSCSI-manager- 飞牛iSCSI挂载工具

## 1. 项目简介

  

iSCSI挂载工具是一个基于Web的iSCSI目标管理应用，提供以下核心功能：

  

- **iSCSI目标发现**：手动发现和自动扫描局域网

- **会话管理**：登录、注销iSCSI目标，支持CHAP认证

- **设备管理**：挂载、卸载、格式化磁盘设备

- **开机自启**：配置iSCSI目标开机自动连接


**访问地址**：`http://<服务器IP>:8080`

  

---

  

## 2. 快速开始

  

### 2.1 首次使用

  

1. **打开应用**

   - 访问 `http://<服务器IP>:8080`

   - 或点击桌面快捷方式"iSCSI挂载工具"

  

2. **检查服务状态**

   - 首页显示iSCSI服务可用状态

  

3. **发现iSCSI目标**

   - 点击"自动发现"按钮扫描局域网

   - 或手动输入目标IP地址

  

4. **登录目标**

   - 选择目标，输入认证信息（如需要）

   - 点击"登录目标"
 
   **飞牛用户到这一步就能在飞牛系统使用iSCSI磁盘了**

5. **挂载磁盘**

   - 在"挂载磁盘设备"区域选择设备

   - 输入挂载点路径（如：`/mnt/iscsi`）

   - 点击"挂载"
