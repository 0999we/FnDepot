# Word模板批量填充工具 - 使用手册

## 📋 功能使用说明

### 1. 文件管理功能

**如何使用**：
1. 点击左侧"文件管理"按钮
2. 在弹出的文件管理窗口中，可以：
   - 查看所有已上传的文件（模板文件、数据文件、输出文件）
   - 点击"上传文件"按钮批量上传多个文件
   - 使用排序下拉框按文件名、时间、大小排序
   - 点击文件右侧的按钮进行删除、重命名、查看详情

**批量上传**：
- 点击"上传文件"按钮
- 在文件选择对话框中按住Ctrl或Cmd键选择多个文件
- 系统会自动识别文件类型并上传
- 上传完成后会自动选择模板和数据文件

### 2. 上传模板功能

**如何使用**：

**方法一：快速选择**
1. 在"步骤1：上传Word模板"页面
2. 点击"从已上传的文件中选择"下拉框
3. 选择已上传的Word模板文件

**方法二：上传新模板**
1. 在文件管理页面点击上传按钮
2. 选择Word文档（.docx格式）
3. 系统自动检测模板中的占位符
4. 查看检测到的占位符列表
5. 点击"下一步"继续

**占位符格式**：
- 使用 `{{字段名}}` 格式
- 例如：`{{姓名}}`、`{{日期}}`、`{{产品名称}}`

### 3. 上传数据功能

**如何使用**：

**方法一：快速选择**
1. 在"步骤2：上传数据文件"页面
2. 点击"从已上传的文件中选择"下拉框
3. 选择已上传的Excel或CSV文件

**方法二：上传新数据**
1. 点击上传区域或"📤"按钮
2. 选择Excel文件（.xlsx、.xls）或CSV文件
3. 系统自动识别数据列
4. 查看数据预览
5. 确认数据行数和列数
6. 点击"下一步"继续

**数据文件格式**：
- 第一行应为列标题
- 每列对应一个数据字段
- 每行为一条数据记录

### 4. 字段映射功能

**如何使用**：
1. 进入"步骤3：映射字段"页面
2. 系统自动匹配占位符和数据列
3. 查看映射结果表格
4. 如需调整，点击下拉框手动选择正确的数据列
5. 确保所有占位符都已正确映射
6. 点击"下一步"继续

**映射说明**：
- 左侧显示模板中的占位符
- 右侧显示对应的数据列
- 绿色表示自动映射成功
- 红色表示需要手动映射

### 5. 生成文档功能

**如何使用**：
1. 进入"步骤4：生成文档"页面
2. 选择生成方式：
   - **生成单个文档**：每条数据生成一个独立的Word文档
   - **合并为一个文档**：将所有生成的文档合并为一个文件
3. 点击"生成文档"按钮
4. 等待生成完成（显示进度条）
5. 生成完成后点击"下载"按钮
6. 选择下载单个文档或批量下载

**生成选项**：
- 可以选择生成部分数据（指定行范围）
- 可以选择是否合并文档
- 支持下载单个或全部文档

## 📝 基本操作流程

### 完整操作步骤

**步骤1：准备模板文件**
1. 创建Word文档（.docx格式）
2. 在需要填充数据的位置插入占位符
3. 占位符格式：`{{字段名}}`

**示例**：
```
检验报告

产品名称：{{产品名称}}
型号规格：{{型号规格}}
检验日期：{{检验日期}}
检验结果：{{检验结果}}
```

**步骤2：准备数据文件**
1. 创建Excel或CSV文件
2. 第一行为列标题（与模板占位符对应）
3. 每行为一条数据记录


**步骤3：上传文件**
- 使用文件管理批量上传，或分步上传模板和数据

**步骤4：映射字段**
- 检查自动映射结果，手动调整错误的映射

**步骤5：生成文档**
- 选择生成方式，点击生成，等待完成

**步骤6：下载文档**
- 下载生成的文档到本地

---



## 💡 使用技巧

1. **批量上传**：可以同时选择多个模板和数据文件上传，系统会自动识别并选择
2. **文件管理**：定期清理不需要的文件，保持文件列表整洁
3. **模板复用**：上传的模板可以重复使用，无需每次重新上传
4. **数据预览**：上传数据后可以预览前几行数据，双击预览的数据可以简单编辑，确保数据正确
5. **分批生成**：数据量大时，可以分批生成，每次处理100-200条数据
